程式使用說明：
點開exe檔案後可選擇模式，可遊玩的模式有兩個：(1) File mode, (2) User Input mode

遊戲說明：
(1) File mode
	用於助教檢測test case，進入後要先輸入檔案資訊，以進行測試輸出
	檔案輸入範例：case.txt

(2) User Input mode
	用於RPG遊戲，進入後不需要輸入檔案資訊，只要數入2即可
	在遊戲進行中進行移動角色，共有四關1~4可進行戰鬥。

[注意事項1]：User Input mode的檔案位置以及名稱
- 用於道館以及玩家
	.\GameData.txt
	.\GameData1.txt
	.\GameData2.txt
	.\GameData3.txt
	.\GameData4.txt

- 用於寶可夢資訊
	.\MoveLib.txt
	.\PokemonLib.txt
	

[注意事項2]：檔案路徑
有關於音樂檔案的路徑為：
	.\Sound\BackgroundMusic
	.\Sound\BattleMusic
